# Sunrise

Website for Country Sunrise Grill & BBQ - Tarboro, NC.

## Getting Started

Install dependencies, then run the dev server:

```bash
npm install
npm run dev
```

Open the URL Vite prints in the terminal (defaults to http://127.0.0.1:5173).

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Pages

- `/` - Home
- `/menu` - Full menu
- `/privacy` - Privacy Policy
- `/terms` - Terms of Service

## Branches

- `main` - production
- `staging` - staging / testing
- `local` - local development
